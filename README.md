# nvda-SessionTimer

Session Timer is an NVDA add-on that announces how long NVDA has been running since startup.

## Features

- Announces full session time.
- Announces short session time.
- Announces NVDA startup time.
- Resets the session timer.
- Announces add-on version information.
- Adds an About item to the NVDA tools menu.
- Includes English and Russian messages.

## Default gestures

- `NVDA+Windows+N`: announce full session time
- `NVDA+Windows+Shift+N`: announce short session time
- `NVDA+Windows+Shift+T`: announce NVDA startup time
- `NVDA+Windows+R`: reset timer
- `NVDA+Windows+I`: announce add-on info
